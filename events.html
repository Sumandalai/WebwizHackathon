API- The Postman of Web Information:

Webwiz, in collaboration with Postman, had organized a super informative, hands-on session on API, called APIs 101, to help students and techno-worms learn API essentials. API or Application Programming Interface is a framework that encapsulates more complex and more used codes and gives a more straightforward syntax for its utilization, allowing faster communication for servers. This event, held towards the end of March, was significant for Webwiz as a club as it got to work together with Postman, which is currently the world's leading collaboration platform for API development.
